# truelist-n8n

[![Free tier](https://img.shields.io/badge/free_plan-100_validations-4A7C59?style=flat-square)](https://truelist.io/pricing)

n8n community node for [Truelist](https://truelist.io) email validation. Validate email addresses directly from your n8n workflows.

> **Start free** — 100 validations + 10 enhanced credits, no credit card required.
> [Get your API key →](https://app.truelist.io/signup?utm_source=github&utm_medium=readme&utm_campaign=free-plan&utm_content=truelist-n8n)
> Test your workflow with the free plan's 100 validations.

## Installation

In your n8n instance, go to **Settings > Community Nodes** and install:

```
truelist-n8n
```

## Configuration

1. Add a **Truelist API** credential in n8n
2. Paste your API key (get one at [truelist.io](https://app.truelist.io/signup?utm_source=github&utm_medium=readme&utm_campaign=free-plan&utm_content=truelist-n8n))
3. Use the **Truelist** node in your workflows

## Getting Started

Sign up for a [free Truelist account](https://app.truelist.io/signup?utm_source=github&utm_medium=readme&utm_campaign=free-plan&utm_content=truelist-n8n) to get your API key. The free plan includes 100 validations and 10 enhanced credits — no credit card required.
